**U-ClanShop** is a modern university e-commerce platform designed to provide users with a seamless online shopping experience. The website enables users to browse products, manage carts, place orders, and explore products through a responsive interface.

## 🚀 Features

* **🔎 Product Search** – Easily browse, and search products.
* **🛒 Shopping Cart System** – Add, update, and remove products from the cart seamlessly.
* **💳 Checkout Workflow** – Smooth and user-friendly checkout process for completing purchases.
* **📱 Responsive Design** – Optimized for desktops, and laptops.
* **⚡ Simple & Fast User Interface** – Clean, lightweight design for a smooth shopping experience.

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP 
* **Database:** MySQL
* **Code Editor:** Visual Studio Code
  
## 📱 How It Works

1. Users create an account or log in.
2. Browse and search available products.
3. Add desired products to the cart.
4. Proceed to checkout.
5. Manage orders and purchase history.

## 🎯 Objective

The objective of **U-ClanShop** is to develop a student-friendly and scalable e-commerce platform that demonstrates core concepts of web development, database management, and online shopping systems.
